# FastTechno Landing Page

Single-page marketing website for FastTechno.

## Tech
- Plain HTML + CSS + JavaScript
- No build step required

## Project Structure
- `index.html` - main page, styles, and scripts
- `media/` - all images and videos used by the page

## Run Locally
1. Open `index.html` in your browser, or
2. Serve the folder with any static server.

## Deploy
Push the `main` branch to GitHub and enable GitHub Pages (if needed).
